assnatapi - un API pour l'Assemblée nationale du Québec

Un API de type REST en logiciel libre pour permettre l'exploitation par programmation des données de l'Assemblée nationale du Québec.

S'informer

Consulter la documentation complète de l'API.

Accéder

L'API est en lecture seule et ne nécessite aucune clef pour le moment. Il s'agit d'accéder aux routes voulues pour obtenir les données:

  • Liste des députés
  • Liste des séances
  • Liste des interventions

Consulter la documentation complète pour plus de détails.
